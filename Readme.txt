Por medio de este proyecto, se intentará programar y diseñar una pagína de una institución educativa de nivel inicial. Se comenzará creando un formulario de datos para las inscripciones de niños a las distintas salas mediante un simulador. Se irán sumando diversos elementos a medida que se sumen distintos conocimientos en las diversas instansias del grupo. Se elige este proyecto ya que trabajo en este espacio educativo por lo que tengo acceso a la información como también a las necesidades que la misma.
